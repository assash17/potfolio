# potfolio

https://assash17.github.io/potfolio/

# Description

간단하게 구성해본 포트폴리오

JS, HTML CSS로 기본 구성

webpack 연습

- file-loader (이미지파일), css-loader, sass-loader 사용
- HtmlWebpackPlugin, MiniCssExtractPlugin, CleanWebpackPlugin 사용

반응형 layout 연습

- flex, rem ...

build

```
yarn build
```

dev용 테스트(webpack-dev server)

```
yarn dev
```
